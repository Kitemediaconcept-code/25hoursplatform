# 25 Hours — Practitioner-First EdTech Platform

A premium, production-ready, SEO-optimized e-learning ecosystem built using modern, responsive styling frameworks, vanilla ES6 JavaScript, and lightweight semantic layouts.

## 🚀 Key Architectural Features
- **Dual Visual Themes**: Real-time adaptive dark/light design system matching user preferences.
- **100% Core Web Vitals Optimized**: Zero external heavy libraries, frameworks, or unnecessary dependencies. Fast load times, lazy-loaded structures, and smooth micro-animations.
- **High-Fidelity SEO Integration**: Structured JSON-LD schemas (FAQPage, Course, AboutPage, BlogPosting, and CollectionPage) embedded directly on targeted directories to optimize Google rich search results.
- **Fluid Layouts**: Full viewport responsiveness for mobile, tablet, and wide-screen desktop displays.

## 📂 Page Directory
The project includes 15 highly integrated workspace templates:
1. `index.html` - Premium interactive landing experience.
2. `courses.html` - Multi-criteria searchable course catalog.
3. `course-details.html` - Syllabus structures with video embeds and faq widgets.
4. `dashboard.html` - Student space featuring enrollment metrics and certificates (Robots Protected).
5. `categories.html` - Domain tracks hub.
6. `pricing.html` - Flexible membership toggles.
7. `about.html` - Our core philosophy and story.
8. `instructors.html` - Directory of active practitioners.
9. `blog.html` - Publications grid view.
10. `blog-details.html` - Code-friendly article presentation.
11. `contact.html` - Form integration with micro-feedback.
12. `login.html` - Dual transition sign-in and sign-up gates.
13. `faq.html` - Expanding accordion support page.
14. `terms.html` - Interactive legal policy center.
15. `404.html` - Animated error view (Robots Protected).

## 🛠️ Getting Started
Simply open `index.html` in any modern web browser or serve it using a lightweight server (e.g., Live Server, Nginx, or node-static). All assets are located within `/assets/css/` and `/assets/js/`.
